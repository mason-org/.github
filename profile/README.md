# About
Package manager for Neovim and Emacs. Easily install and manage LSP servers, DAP servers, linters, and formatters. Easily install and manage LSP servers, DAP servers, linters, and formatters.

For a list of all available packages in the core registry, refer to the [Package list][package list].

For more information about the Neovim plugin, `mason.nvim`, refer to [mason-org/mason.nvim][mason.nvim].

For more information about the Emacs plugin, `mason.el`, refer to [mason-org/mason.el][mason.el].
